暗記の小屋のガイドページ

知りたい箇所をクリックすることで解説を見られます。

暗記の小屋で投稿されたすべての問題集がこのページに集結されます。

問題集の左下にある星マークは、お気に入りの印で、どれくらいお気に入りがついているのかが分かります。

編集ボタンをクリックすると誰かが作った問題集を編集することができます。改変履歴は残るので、許可は必要ありません。

問題集のつくり方

ページ右上の「問題集を作る」をクリックし、問題作りのガイドページを参考にしてください。

セーブボタンをクリックしたら、サイトを離れてもデータが保持されます。

セーブボタンの例：　

「 ⋮ 」をクリックすると、二つのボタンが現れます。
参考図↓

1. まず以下のようなファイルを格納したフォルダを作成します。（ファイルは複数個あっても構いません。入れ子構造も対応しています。）

test.json
{
    contents: [
        ["問題１", "解答１"],
        ["問題２", "解答２"],
        ["問題３", "解答３"]…
    ],
    shuffle: true
}

contentsの配列の要素は、1つ目が表示される問題、2つ目がその答えになるようにする。

shuffleの値はtrueにするとシャッフルが行われ、falseにすると、シャッフルが自動で防がれます。

2. フォルダ選択のページ にアクセスして、「フォルダを選択」ボタンをクリックする。

3. 1で作ったフォルダをアップロードする。
